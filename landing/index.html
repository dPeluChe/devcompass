GitHub is great at one repo. devcompass is great at all of them.

GitHub is fast for repository detail. It is slow for the questions developers actually ask every morning:

  • Which of my repos got activity yesterday?
  • Where are PRs waiting on me?
  • Which open PRs across all my orgs have failing CI?
  • Did my token authorize the org I expected, or did SSO silently block it?
  • Can I reload the page without waiting 30 seconds for a full GitHub sync?

devcompass answers those from one dense, fast UI.

What github.com gives you

  • Notifications inbox (one event at a time)
  • Pinned repos on your profile
  • Per-repo dashboards
  • Search if you know what to type

What devcompass adds

  • Cross-org PR inbox — every PR waiting on you, all orgs, one list
  • Operational digest — heatmap + 24h/7d/30d activity snapshot
  • Since-last-visit diff — what changed while you were away
  • Local-first cache — reloads paint instantly, no 30-second wait

Up and running in 30 seconds

No signup. No backend account. No credit card. Bring a GitHub PAT and go.

  1. 1

    Open the app

    Use the hosted demo or run npm run dev on your own machine. The dist/ folder is a complete deployable bundle — no server needed.

  2. 2

    Paste your PAT

    A GitHub classic Personal Access Token with repo + read:org. The token is stored in your browser’s localStorage, never sent to any server we operate.

  3. 3

    See everything

    The Digest scope loads your activity snapshot, heatmap, and the PRs waiting on you. Reloads are instant from then on — IndexedDB caches everything locally.

Privacy is a feature, not a default

devcompass is a browser-only single-page app. There is no backend operator. There is no analytics. There is no third-party network call outside the GitHub API itself.

WhatWhere it livesWhy
Your GitHub PAT localStorage Sent only to api.github.com as a bearer token. Never to a server we operate.
Cached repos & PRs IndexedDB (ghviewer DB) Lets the UI render instantly on reload before the next sync finishes. TTL-bound and auto-pruned.
Per-org visibility flags localStorage Remembers which orgs you toggled off in Settings.

Everything lives on your machine. Settings → Storage → Clear all cache wipes it whenever you want.
